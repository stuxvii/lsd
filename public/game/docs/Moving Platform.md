The Moving Platform will move across a Curve's points that you created.
<br>
To create a Curve, click "Create Curve" while having your Moving Platform selected.
<br>
You should now see all these fancy tools:
<br>
![alt text](/game/docs/image-12.png)
<br>
<br>
* Select Points: self-descriptive
<br>
* Select Control Points: Allows you to freely move points
<br>
* Select Tilt Handles: self-descriptive
<br>
* Add Point: Allows you to add a point anywhere with your pointing device
<br>
* Delete Point: delete point with pointing device
<br>
* Close Curve: Creates a point in the location of the first added point, to close the Curve
<br>
* Clear Points: Deletes all points
<br>
<br>
**REMEMBER TO CLOSE THE CURVE AFTER YOU CREATED IT OR THE PLATFORM 
<br> 
WILL TELEPORT BACK TO ITS STARTING POSITION ON THE END OF THE CURVE**
<br>
